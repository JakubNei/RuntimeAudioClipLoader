AudioClip

Audio data will be loaded into this UnityEngine.AudioClip instance. Make sure to call UnityEngine.Object.Destroy(UnityEngine.Object) on the instance after you are done using it. If you have thousands of unused AudioClips you risk running out of memory if you don't destroy them.

Load(Stream, SelectDecoder, string, bool, bool, bool)

Loads an AudioClip from Stream, you need to manually supply AudioFormat. Use GetAudioFormat to get AudioFormat from filePath. Supported formats: waw, mp3, aiff, ogg.

Declaration

Stream used to read the audio data
SelectDecoder audioFormat

Use GetAudioFormat to get AudioFormat from filePath
string unityAudioClipName

Used as Unity's name of the AudioClip
bool doStream

audioClip will be loaded and decoded on the fly on demand, use for long one time use clips

Load(string, bool, bool, bool)

Loads an AudioClip from file located at filePath. Supported formats: waw, mp3, aiff, ogg.

Declaration
public static AudioClip Load(string filePath, bool doStream = false, bool loadInBackground = true, bool useCache = true)
Parameters
Type Name Description
string filePath

filePath is used as Unity's name of the AudioClip and as a key used in the caching dictionary
bool doStream

audioClip will be loaded and decoded on the fly on demand, use for long one time use clips
bool loadInBackground

if !doStream and loadInBackground then loading is done in own thread so it doesnt hang up caller's thread
bool useCache

audioClip will be cached, so it won't have to be loaded again next time
Returns
Type Description
UnityEngine.AudioClip

null if error (i.e. unsupported format, invalid audio file), may contain undefined data or only silence if clip is not yet loaded

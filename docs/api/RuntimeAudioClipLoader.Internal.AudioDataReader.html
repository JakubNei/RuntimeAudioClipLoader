Class AudioDataReader

This is a custom modified NAudio.Wave.AudioFileReader. Has added support for Ogg with NVorbis.NAudioSupport.VorbisWaveReader Has addes support for mannaged MP3 with NLayer.NAudioSupport.Mp3FrameDecompressor
